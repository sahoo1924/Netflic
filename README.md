# Netflic
## 📌 About the Project

This project is a front-end clone of Netflix, designed to practice and demonstrate skills in web development, UI design, and responsive layouts.

The application mimics the look and feel of Netflix, including movie sections, banners, and interactive UI elements.

---

## ✨ Features

- 🎥 Netflix-style homepage layout  
- 📱 Fully responsive design  
- 🎞️ Movie/series sections with thumbnails  
- 🎨 Clean and modern UI  
- ⚡ Smooth user experience  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 / SCSS  
- JavaScript
